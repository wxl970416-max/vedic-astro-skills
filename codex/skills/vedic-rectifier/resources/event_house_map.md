## 匹配判定规则

### 强匹配 (✅✅)
```
大运/小运主星 = 事件对应宫位的宫主
→ "Venus大运期间结婚，Venus恰好是7宫主" = 强匹配
```

### 中匹配 (✅)
```
大运/小运主星落在事件对应宫位
→ "Jupiter大运期间结婚，Jupiter落在7宫" = 中匹配

大运/小运主星照射事件对应宫位
→ "Jupiter大运期间结婚，Jupiter从1宫照射7宫" = 中匹配

大运/小运主星 = 事件的Karaka
→ "Venus大运期间结婚，Venus是自然婚姻Karaka" = 中匹配
```

### 弱匹配 (⚠️)
```
大运/小运主星与事件宫位的宫主有相位
→ 间接关联，可以接受但权重低
```

### 不匹配 (❌)
```
大运/小运主星与事件宫位无任何关联
→ 强烈暗示时间可能偏差（宫主表错了）
```

## 逆推逻辑

当事件不匹配时，用以下逻辑推理时间偏移方向：

```
已知: 
  事件=结婚, 大运主星=Mars
  当前: Mars管L3和L8 → 与7宫无关 → ❌

逆推:
  Mars管什么星座？→ Aries和Scorpio
  哪个Lagna能让Mars管7宫？
    → Lagna=Aries → 7宫=Libra (Venus管) → Mars不管7宫
    → Lagna=Taurus → 7宫=Scorpio → Mars管7宫 ✅
    → Lagna=Libra → 7宫=Aries → Mars管7宫 ✅
  
  如果当前Lagna=Gemini → 需要向Taurus方向移动
  → 时间要往回调（Lagna度数减小）
```

### 使用规则

1. **权重**：物理标记匹配 = 2倍权重（vs 主观描述1倍）
2. **双盘对比最强判据**：两个Lagna候选的Mars落宫不同时，物理标记是最快的区分方式
3. **盘外验证必含**：Step 5盘外验证中至少包含1条物理标记预测
4. **措辞注意**：使用"是否有"而非"您一定有"，Mars指示倾向而非绝对必然
